数字分类

问题描述：
有一些数字取值范围在[0,100]，分别找出[0,50]，(50,100]的数学，进行分类。

输入：
n个数字，排成一行，用","分隔。
系统会默认生成data对象，保存测试数据集。

注意：代码中请不要写入data对象的赋值代码。


输出：
[0,50]设置分类0,(50,100]分类为1

注意：代码中结果一定要放在answer对象中，其他输出则无效。

样例输入：
76,88,62,26,85,43,38,46,22,7

样例输出：
> answer
 [1] 1 1 1 0 1 0 0 0 0 0


R参考代码实现：

例一：
answer<-rep(0,length(data))
answer[which(data>=50)]<-1
